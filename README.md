Programa de consola que permite interactuar a través de un chat con GPT-3.
Debe generar tu api key y reemplazarla en el archivo config.json, también puedes cambiar el modelo a usar.

MODELO				DESCRIPCIÓN
text-davinci-003	El modelo GPT-3 más capaz. Puede realizar cualquier tarea que puedan realizar los otros modelos, 
					a menudo con mayor calidad, mayor duración y mejor seguimiento de instrucciones. 
					También admite la inserción de finalizaciones dentro del texto.	4,000 tokens
text-curie-001		Muy capaz, pero más rápido y de menor costo que Davinci.
text-babbage-001	Capaz de realizar tareas sencillas, muy rápidas y de menor costo.
text-ada-001	    Capaz de tareas muy simples, generalmente el modelo más rápido de la serie GPT-3 y el costo más bajo.
URL: https://beta.openai.com/docs/models/gpt-3
